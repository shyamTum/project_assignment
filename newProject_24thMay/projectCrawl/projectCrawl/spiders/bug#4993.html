tobiasKaminsky commented Jan 22, 2020

To simplify this whole mess, I think it is best to have each Activity in its own separate life cycle:

  • start app
  • FDA checks if account is available
    • no account: start FirstRunWizard (without any dependencies), that's it
    • account exists: continue to FDA
  • FirstRunWizard: starts login or choose provider, ends itselfs
  • Login / choose provider: go through login or start FirstRunWizard again
  • if login is successful go to FDA

This way we only have one Activity at the same time running.

Additional we have to consider:

  • click on nc://login/ link which currently targets FDA
  • scan QR code which is basically the same link, also targets FDA
